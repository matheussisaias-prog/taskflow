📱 Configurar WhatsApp (CallMeBot)

1 Salve o número +34 644 44 21 48 nos seus contatos com o nome CallMeBot
2 Abra o WhatsApp e envie exatamente: I allow callmebot to send me messages para esse contato. Este passo é obrigatório — sem ele a API Key não funciona.
3 O CallMeBot responderá com: "Your APIKEY is 123456" — copie apenas os números
4 No campo Número abaixo, use DDI+DDD+número sem espaços ou símbolos.
Exemplo Brasil: 5585999991234
5 Cole a API Key, clique Salvar e depois 🧪 Testar envio para confirmar.
⚠ Atenção: O CallMeBot é um serviço gratuito de terceiros. Se a mensagem não chegar, refaça o passo 2 — a ativação expira ocasionalmente.
